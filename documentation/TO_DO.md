### PARTIE BASIQUE OBLIOGATOIRE (SANS MODULES)

##### GENERAL:
	* Notre site internet Devrait-etre une [single page application SPA](https://en.wikipedia.org/wiki/Single-page_application). ET les boutons (avant/arriere) devrait etre functionnels.



##### BACKEND:
	* Utiliser `PHP` sans framework (peut etre remplacer par le module Framework)
	* En cas d'utilisation d'une Base de donnees, ca doit suivre le module Database

##### FRONTEND:
	* Utiliser `Typescript` pour le front (peut etre modifier par le module Front)

##### DOCKER:
	* Notre site devrait etre entierment hebergé dans un Docker et  lancé avec une seule command.

> **ALERT:** En vue des limitations sur du hub 42. Nous avons la solution de porter le projet dans une Machine virtuelle pour avoir tout les doits et la liberte d'installer les version souhaitable!


##### AUTRES:
	* Notre site doit etre compatible avec la derniere version de FireFox et les autres Navigateurs.
	* Notre site NE devrait PAS y a voir aucune anomalie durant l'usage.
